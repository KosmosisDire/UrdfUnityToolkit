# URDF Toolkit
Unity package for importing URDF/SRDF robot descriptions and converting Xacro files.

## Add to Unity

1. Open **Window > Package Manager**.
2. Select **+ > Add package from git URL...**.
3. Paste:
```
https://github.com/KosmosisDire/UrdfUnityToolkit.git
```
   
Or add this to your `Packages/manifest.json`:
```json
"com.kosmosisdire.urdftoolkit": "https://github.com/KosmosisDire/UrdfUnityToolkit.git"
```
